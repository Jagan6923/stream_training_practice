Meta and Link Tags

Meta tags provide document information, while link tags connect the page to external resources.

Why these tags matter

  • charset sets the character encoding
  • viewport supports responsive layouts
  • description improves page summaries
  • link connects stylesheets, icons, and resources

Resources used here

The stylesheet provides the shared lesson layout, the favicon adds a browser tab icon, and prefetch tells the browser that it may need a resource soon.

Concepts covered: metadata, favicon, stylesheet links, and prefetch.
